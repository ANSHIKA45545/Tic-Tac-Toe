#Tic-Tac-Toe

This is a simple **Tic-Tac-Toe game** implemented in Java.  
The game is played in the **console** by two players: `X` and `O`.  
Players take turns to place their marks on the 3Ã—3 board until one of them wins or the game ends in a draw.
